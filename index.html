vagrant.docker

Development with docker the easy way

Overview

A vagrant box based on Ubuntu for developing/working with Docker.

Dependencies

  • Ansible
  • Virtualenv
  • VirtualBox [tested with version 4.3.20]
  • Pip-Tools
  • Vagrant-Hostsupdater
  • Vagrant-Vbguest

Install

Clone this repo

git clone https://github.com/leftxs/vagrant.docker

Run Makefile

make install

Documentation built with MkDocs.
